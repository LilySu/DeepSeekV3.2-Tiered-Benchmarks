DeepSeek-V3 671B Architecture

arXiv: 2412.19437 | 671B total parameters, ~37B active per token

Model Overview

DeepSeek-V3 is a Mixture of Experts (MoE) language model with three key innovations:

MLA - Multi-head Latent Attention for KV cache compression

Grouped MoE - 256 experts with grouped top-K routing

MTP - Multi-Token Prediction for speculative decoding

671B params 61 layers 7168 hidden 128 heads 129K vocab

Layer Structure

Embedding (129280 x 7168)
Dense Layer 0-2: MLA + SwiGLU FFN (4x)
MoE Layer 3-60: MLA + MoE FFN (256 experts)
RMSNorm + LM Head (7168 x 129280)
MTP Head (1 layer)

3 dense + 58 MoE = 61 total layers
